How to run this flask web app:
1. install anaconda on your computer
2. install vscode
3. on vscode add extension "Python"
4. Click view>Command Palette>Python: Select Interpreter>choose the recommended python version
5. open folder "gandaberunda" on vscode
6. open app.py
7. run it by clicking triangle shape on the right top corner (Run Python File)
8. go to "127.0.0.1:8000/views" on your browser
